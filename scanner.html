Scanner.bat (SFC: verificar y reparar)

El Scanner es un submenú que ejecuta SFC (System File Checker). Sirve para comprobar si Windows tiene archivos dañados y, si hace falta, intentar repararlos.

⬇ Descargar Scanner.bat Volver a Mantenimiento

Recomendación: ejecutar la consola como administrador para evitar errores de permisos.

Qué hace cada opción

1 · Verificar

Comprobar sin reparar

Comando: sfc /VERIFYONLY. Revisa integridad de archivos protegidos del sistema sin hacer cambios.

2 · Reparar

Comprobar y reparar

Comando: sfc /SCANNOW. Puede tardar bastante. Intenta reemplazar archivos dañados.

3 · Volver

Salir del Scanner

El script termina con exit /b para volver al mantenimiento (si fue abierto con call Scanner.bat). Así cumple “se abre y vuelve bien”.

Integración correcta con el mantenimiento

Para que el flujo sea limpio:

:: En MantenimientoV.2.bat
:abrir_scanner
call Scanner.bat
goto inicio

:: En Scanner.bat (al salir)
exit /b
