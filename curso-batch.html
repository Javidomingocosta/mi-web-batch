Historia rápida: de MS-DOS a Batch

Los archivos .BAT nacen como guiones de comandos en la era DOS: una forma de automatizar tareas repetitivas (copias, menús, configuraciones). Hoy siguen funcionando en Windows a través de cmd.exe.

Años 80

MS-DOS y el intérprete de comandos

Comandos como DIR, CD, COPY. Los .BAT encadenan comandos como si fuera una receta.

Años 90–2000

Windows mantiene compatibilidad

Batch sigue disponible para administración básica y scripts rápidos.

Hoy

Automatización ligera

Ideal para menús, mantenimiento escolar, lanzadores y tareas simples. Para cosas avanzadas existe PowerShell, pero Batch sigue siendo rápido y universal en Windows.

Consejo: crea una carpeta “PruebasBatch” y ejecuta ahí tus scripts para no borrar archivos por error.

Módulo 5 — Menús (base para el mantenimiento y el juego)

Conceptos: choice errorlevel goto

@echo off
:menu
cls
echo 1) Opcion A
echo 2) Opcion B
echo 3) Salir
choice /c 123 /n /m "Elige: "
if errorlevel 3 goto fin
if errorlevel 2 goto b
if errorlevel 1 goto a

:a
echo Elegiste A
pause
goto menu

:b
echo Elegiste B
pause
goto menu

:fin
echo Adios
pause
