¿Qué es un Sistema Operativo?

Un sistema operativo (SO) es el “controlador central” que gestiona recursos (CPU, memoria, disco, periféricos) y ofrece servicios para que las aplicaciones funcionen. Es la interfaz real entre hardware y software.

¿Qué hace exactamente?

Gestión de procesos

Decide qué programa usa la CPU, cuándo y cuánto tiempo (planificación).

Gestión de memoria

Asigna RAM, evita choques entre programas y usa memoria virtual si hace falta.

Sistema de archivos

Organiza datos en carpetas/archivos y controla permisos, rutas y operaciones de lectura/escritura.

Control de dispositivos

Habla con teclado, pantalla, impresora, red… mediante controladores (drivers).

Viaje rápido en el tiempo (timeline)

Una línea temporal simplificada para ubicar cómo llegamos a sistemas modernos:

Años 50–60 · Mainframes

Procesamiento por lotes

Trabajos “en cola”: se ejecutan uno detrás de otro. La interacción era mínima.

Años 70 · Multiusuario

Tiempo compartido

Varios usuarios usan el mismo sistema a la vez. Terminales, cuentas, permisos.

Años 80 · PC

SO para ordenadores personales

Arranca la era del PC: sistemas más simples, orientados a un usuario y hardware limitado.

Años 90–hoy

Interfaces gráficas + redes

GUI, multitarea real, controladores complejos, internet, seguridad y actualizaciones constantes.
